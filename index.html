NightOut-iOS

Repository for NightOut iOS app A demo of the current version of NightOut App, a table reservation app for night clubs made for DinoMedia.

Demo of the App

  • https://youtu.be/VtYXMIx-GZQ

Requirements

  1. Download FacebookSDK here https://developers.facebook.com/docs/ios/
  2. Make sure it's under ~/Documents/FacebookSDK
  3. CocoaPods
  4. Do pod install in the root directory in Terminal
